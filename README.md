#  Car Price Optimization & Market Analysis

##  Business Problem
The automotive industry is evolving rapidly with increasing competition and changing consumer demand. This project aims to analyze how vehicle features influence pricing and demand, and how manufacturers can optimize pricing and product strategies.

##  Objectives
- Analyze factors affecting car prices
- Understand consumer demand patterns
- Perform market segmentation
- Build a predictive model for pricing
- Create an interactive dashboard

##  Dataset
- 11,159 rows
- 16 features
- Source: Kaggle (Car Features and MSRP)

- ##  Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI

##  Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Business Analysis
4. Feature Engineering
5. Machine Learning
6. Dashboard Creation
